- Currency Adder

# Getting Started
## Prerequisites
- Basic knowledge of C++ and reverse engineering!
- Familiarity with game hacking concepts such as memory manipulation and hooking.

## Setup
**1.** Clone the repository.

**2.** Build in Release x64 / Debug x64

**3.** Inject!
